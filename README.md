# ogpng

Engineer 向け OGP シェアサービス。

ソースコードの仕組みや実装の解説は、 [ソースコードから OGP を生成しシェアする Ogp as a Service を作った(そして飽きたのでコードを公開する)](http://localhost:8000/share-ogp) をご覧ください。

## firebase 周り

ぼくの APIKEY を埋め込んでいますが、無料プランなので使いすぎると止まります。
自分の鍵でお試しください。

## Dev

```sh
yarn install

yarn develop
```

## 機能

- Editor で 画面を作れる
- 画面を画像として保存して OGP 化
- 認証
